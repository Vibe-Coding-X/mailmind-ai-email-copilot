# MailMind v0.7.1 Release Pipeline

v0.7.1 stabilizes the desktop release pipeline for Windows, macOS, and Linux.

## Added

- Updated GitHub Actions desktop workflow to current action versions and Node 24.
- Verified three-platform packaging on Windows, macOS, and Linux.
- Tag-triggered GitHub Release draft flow for desktop installers.
- Clearer documentation for GitHub Actions artifact downloads and packaged installers.

## Changed

- Release documentation now explains that GitHub Actions artifacts are downloaded as `.zip` archives.
- Desktop roadmap versions are shifted to `v0.7.2`, `v0.7.3`, `v0.7.4`, and `v0.8.0`.

## Verified

- Windows artifact downloads and installs successfully.
- macOS artifact downloads and installs successfully.
- Linux artifact downloads and installs successfully.

## Not Changed

- The desktop app remains an Electron shell around the MailMind web UI.
- Backend, database, and job queue are still external dependencies.
- The installer does not yet provide an all-in-one local runtime.

## Next

v0.7.2 will focus on desktop UX improvements: tray support, window state persistence, menus, and notifications.
