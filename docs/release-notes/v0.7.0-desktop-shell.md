# MailMind v0.7.0 Desktop Shell

v0.7.0 adds an Electron desktop shell for Windows, macOS, and Linux.

## Added

- Electron desktop application that loads the MailMind web UI.
- Health check on startup with friendly fallback screen when services are unavailable.
- Configurable app URL and health URL via environment variables or config file.
- External links open in the system browser.
- Security-hardened Electron configuration (contextIsolation, no nodeIntegration).
- electron-builder packaging for Windows (NSIS), macOS (DMG), and Linux (AppImage).
- GitHub Actions workflow for automated multi-platform desktop builds.
- Desktop shell documentation (setup, release, security, roadmap).

## Not Included

- Embedded backend, database, or job queue.
- OAuth deep links.
- Auto-update.
- System tray.
- Code signing or notarization.
- All-in-one desktop mode.

## Usage

1. Start MailMind backend and frontend locally.
2. Launch the Electron app.
3. The app loads the web UI at `http://127.0.0.1:3000`.
